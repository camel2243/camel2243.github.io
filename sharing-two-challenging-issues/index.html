Cookie SameSite Attribute

  • Strict
    Cookie 只在 same-site 時才會被發送出去
  • Lax（預設值）
    Cookie 會在 same-site / cross-site top-level navigation 送出
  • None
    Cookie 會在 cross-site / same-site 時都被發送出去

https://datatracker.ietf.org/doc/html/draft-west-first-party-cookies


Cookie SameSite Attribute

                    Chrome 在 2020 年 2 月起針對 Cookie 的發送規則調整

  • Strict
    Cookie 只在 same-site 時才會被發送出去
  • Lax（預設值）
    Cookie 會在 same-site / cross-site top-level navigation 送出
  • None
    Cookie 會在 cross-site / same-site 時都被發送出去

https://datatracker.ietf.org/doc/html/draft-west-first-party-cookies

#1 跨平台快速註冊 - 解法

  1. CodeMentor 設定 CSP：frame-src
  2. Arc 設定 CSP：frame-ancestors
  3. Arc 設定 CORS
  4. Arc 會員登入會帶有 login_token (same-site: none)
  5. Arc 提供一個註冊頁面的 embedding frame code
  6. CodeMentor 打 /api/is_user_logged_in 決定是否塞 Iframe
